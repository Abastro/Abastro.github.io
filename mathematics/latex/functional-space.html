Normed vector spaces

Consider \(X\), a vector space over an analytic field \(F\), i.e., \(F = \mathbb{R}\text{ or } \mathbb{C}\).

Definition 1. \(p : X \to [0, \infty)\) is called a semi-norm if

  1. Subadditive, \(p(x+y) \leq p(x) + p(y)\).

  2. Sublinear, \(p(\lambda x) = \left| \lambda \right| p(x)\) (\(\lambda \geq 0\)).

Definition 2. A semi-norm \(\left\| \text{-} \right\| : X \to [0, \infty)\) is called a norm in \(X\) if \(\left\| x \right\| = 0\) if and only if \(x = 0\).

A vectorspace with a norm is called a normed (vector)space.

Remark 1. A normed space has a canonical metric \(d(x, y) = \left\| x - y \right\|\).

Remark 2. Any subspace of a normed space is clearly a normed space.

Definition 3. A category of normed spaces is given by

  • Objects: Normed spaces \((X, \left\| \text{-} \right\|_X)\).

  • Morphisms: Continuous linear maps \(T: X \to Y\).

The morphisms are called bounded linear maps, for the following reason.

Proposition 1. The following are equivalent for a linear map \(T : X \to Y\) between normed spaces.

  1. \(T\) is continuous.

  2. \(T\) is continuous at \(0\).

  3. \(T\) is bounded, i.e., \(\left\| Tx \right\|_Y \leq c \left\| x \right\|_X\) for a constant \(c > 0\).

Proof. \(\ref{bdd-map:bounded} \Rightarrow \ref{bdd-map:conti}\) is given by Lipschitz continuity, as \(\left\| Tx - Ty \right\|_Y \leq c \left\| x - y \right\|_X\). \(\ref{bdd-map:conti} \Rightarrow \ref{bdd-map:conti-0}\) is obvious. For \(\ref{bdd-map:conti-0} \Rightarrow \ref{bdd-map:bounded}\), notice that a \(\delta\)-ball at \(0 \in X\) should map inside the unit ball in \(Y\). Then, scaling puts a uniform bound at \(\left\| Tx \right\|_Y / \left\| x \right\|_X\). ◻

Note that notion of quotient space carries over from vector spaces.

Proposition 2. If \(M\) is a closed subspace of \(X\), the quotient space \(X/M\) is a normed space with a norm \(\left\| x + M \right\| = \inf_{y \in M} \left\| x+y \right\|\), and the projection \(X \to X/M\) is bounded (by 1).

Definition 4. A Banach space is a normed space which is complete as a metric space.

The following criteria can be used to show a normed space is a Banach space.

Theorem 3. A normed space \(X\) is complete as a metric space if and only if absolutely convergent series in \(X\) converges.

\(L^p\) spaces

\(L^p\) spaces are important examples of Banach spaces.

Fix a complete measure space \((X, \mathcal{M}, \mu)\).

Definition 5. For \(p \in [1, \infty)\), a \(p\)-norm of \(f \in \hom(X, \mathbb{C})\) is \[\left\| f \right\|_p := \left( \int \left| f \right|^p d\mu \right)^{1/p} \in [0, \infty].\] An \(L^p\)-space is, then, \[L^p(X, \mathcal{M}, \mu) := \{ f \in \hom(X, \mathbb{C}) \mid \lVert f \rVert_p < \infty \} / \sim.\] with equality given as \(f \sim g\) when \(f = g\) \(\mu\)-almost everywhere.

Definition 6. A \(\infty\)-norm on \(\hom(X, \mathbb{C})\) is \[\left\| f \right\|_\infty := \inf \{ a \geq 0 : \left| f \right| \leq a \; \mu\text{-a.e.}\}\] Then, \(L^\infty\)-space is defined analogously as above.

To justify the name, \(p\)-norm should be a norm of \(L^p\) space.

Theorem 4 (Hölder inquality). Suppose \(1 < p < \infty\) and \(\frac{1}{p} + \frac{1}{q} = 1\). Then, \(\left\| fg \right\|_1 \leq \left\| f \right\|_p \left\| g \right\|_q\) for \(f, g \in \hom(X, \mathbb{C})\). In particular, if \(f \in L^p\), \(g \in L^q\), then \(fg \in L^1\).

Proof. We may assume \(\left\| f \right\|_p, \left\| g \right\|_q \in (0, \infty)\), as the inequality is obvious on boundary conditions. Then, we may normalize by \(f \leftarrow \frac{f}{\left\| f \right\|_p}, g \leftarrow \frac{g}{\left\| g \right\|_q}\), so that the goal is \(\left\| f \cdot g \right\|_1 \leq 1\). By integrating the following \[\left| f(x)g(x) \right| \leq \frac{1}{p} \left| f(x) \right|^p + \frac{1}{q} \left| g(x) \right|^q,\] one obtains \(\left\| f \cdot g \right\|_1 \leq \left\| f \right\|_p^p / p + \left\| g \right\|_q^q / q = 1\), as desired. ◻

Lemma 5 (Minkowski’s inequality). For all \(1 \leq p < \infty\) and \(f, g \in L^p\), \(\left\| f + g \right\|_p \leq \left\| f \right\|_p + \left\| g \right\|_p\).

Theorem 6. \(L^p\) is a Banach space.
